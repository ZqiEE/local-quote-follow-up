# Local Quote Follow-up RC1

**A local-first lead capture and quote follow-up app for service businesses.**

Find leads. Move them into follow-up. Keep data local.

## What this release is

This release is the first working version of Local Quote Follow-up for small-scale testing.

It connects three steps into one local workflow:

- finding leads that look worth contacting
- moving selected leads into a real pipeline
- pushing quote follow-up forward from one workspace

This is not a generic CRM, and it is not an automated sales robot.

## What you can do with it

With this release, you can:

- search for company websites that look like real prospects
- review which leads are actually worth contacting
- move a lead into follow-up with one click
- let the app create or reuse Customer / Quote / Task
- work from Today Queue to see what to push first
- use Quick Follow-up to keep the quote moving
- review history in Timeline
- undo the latest AI write-back if needed

## Main workflow

1. Start in **Lead Capture**
2. Review the returned leads
3. Click **Move to Follow-up**
4. Let the app write into **Customer / Quote / Task**
5. Continue from **Today Queue**
6. Use **Quick Follow-up**
7. Review **Timeline**
8. Use **Undo Last AI Update** if needed

## Local / private notes

- Data stays on your machine by default
- External AI APIs are off by default
- The local model runtime is visible inside Settings
- You can inspect the data path, model path, and runtime mode yourself

## Install

1. Download the Windows installer
2. Install the app
3. Complete first-run setup
4. Start from Lead Capture or Today Queue

## Current limits

This release is usable, but it is still early.

Current limits:

- lead quality still needs more real-user validation
- the app helps you find and organize leads, but does not contact them for you
- it does not send messages, send emails, or run outbound workflows automatically
- it is not a full autonomous sales agent
- low-frequency pages are not yet fully polished or fully translated

## Best-fit users

This release fits best if you:

- run a service business
- need more qualified companies to look at
- want a local-first workflow
- want to move from lead discovery into quote follow-up without switching tools

## Feedback

The most useful feedback is:

- what search input you used
- how many leads came back
- how many you would actually contact
- which leads you moved into follow-up
- whether they appeared correctly in Today Queue
- what step felt slow, unclear, or not trustworthy
