# Local Quote Follow-up RC1

**一款面向服务型团队的本地优先找客与报价推进软件。**

找客户，推进跟进，把数据留在本机。

## 这一版是什么

这是 Local Quote Follow-up 第一版可运行的 RC 版本，适合做小范围真实测试。

这一版第一次把三件事接成了一条本地主链：

- 找到看起来值得联系的潜在客户官网
- 把选中的 lead 推进进真实 pipeline
- 从一个工作台里持续推进报价跟进

它不是通用 CRM，也不是全自动销售机器人。

## 你现在能用它做什么

这一版你可以：

- 搜到看起来像真实潜在客户的公司官网
- 判断哪些 lead 真值得联系
- 一键把 lead 推进到跟进链路
- 让系统创建或复用 Customer / Quote / Task
- 从 Today Queue 看今天最该先推哪一笔
- 用 Quick Follow-up 持续推进报价
- 在 Timeline 里看历史
- 必要时撤回最近一次 AI 写回

## 主路径

1. 从 **Lead Capture** 开始
2. 查看返回的潜在客户结果
3. 点击 **Move to Follow-up**
4. 让系统写入 **Customer / Quote / Task**
5. 回到 **Today Queue**
6. 使用 **Quick Follow-up**
7. 查看 **Timeline**
8. 必要时使用 **Undo Last AI Update**

## 本地 / 隐私说明

- 数据默认保存在本机
- 默认不连接外部 AI API
- 本地模型运行环境可在 Settings 里直接查看
- 数据路径、模型路径、运行模式都可以自己检查

## 安装

1. 下载 Windows 安装包
2. 安装软件
3. 完成首次启动设置
4. 从 Lead Capture 或 Today Queue 开始使用

## 当前限制

这一版已经可用，但还处在较早阶段。

当前限制：

- lead 质量还需要更多真实用户验证
- 它能帮你找客和推进，但不会替你自动联系客户
- 它不会自动发消息、发邮件或跑外呼流程
- 它不是全自动销售代理
- 低频页面还没有完全打磨，也没有完全翻译完

## 最适合谁

这版最适合：

- 本身就在做服务型业务的人
- 需要更多更像真实客户的公司官网线索的人
- 希望工作流默认留在本地的人
- 想把找客和报价推进接成一条链的人

## 反馈

最有价值的反馈是：

- 你用了什么搜索输入
- 返回了多少 lead
- 其中多少条你真愿意联系
- 你点了哪些进入跟进
- 它们有没有正确进入 Today Queue
- 哪一步最慢、最不清楚、最不放心
